
# Research Project: WSJ Articles Scraping

## Research Question & Data
Collecting 26,000+ Wall Street Journal (WSJ) articles (2010–2024) to build a large-scale dataset for future finance and textual analysis research.

## Method
- *Web Scraping Pipeline:* Automated collection of WSJ articles.  
- *Data Cleaning & Preprocessing:* Preparing text for NLP tasks, including tokenization, stopword removal, and basic text normalization.  

## Tools
- *Languages & Libraries:* Python, Requests, BeautifulSoup, Pandas  
- *Pipeline Overview:* Scraping → Cleaning → Structuring → Ready for NLP/Analysis

## Next Steps
- Implement NLP analysis (e.g., sentiment analysis, keyword extraction, or topic modeling).  
- Explore trends, insights, and patterns in financial news over time.
